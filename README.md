# MarkDown Studio

ローカルのマークダウンファイルを快適に編集できる専用エディター。

## 概要

MarkDown Studioは、ブラウザ上で動作する軽量かつ高機能なマークダウンエディターです。
ローカルファイルの読み込み、編集、保存に対応し、リアルタイムプレビューやアウトライン表示など、快適な執筆環境を提供します。

## 主な機能

- **ファイルの読み書き**: ローカルのマークダウンファイルを直接開いて編集、別名保存や上書き保存が可能。ドラッグ＆ドロップによる読み込みにも対応。
- **リアルタイムプレビュー**: マークダウンの編集内容を右ペインに即座に表示。
- **Mermaid図のレンダリング**: フローチャート、シーケンス図、クラス図など、Mermaid記法で書かれた図を自動的にグラフィカルにレンダリング。
- **リッチなエディター機能**: CodeMirrorベースの高性能エディター。シンタックスハイライト、各種ショートカットキー完備。
- **ツールバー操作**: 見出し、太字、リスト、チェックリスト、引用、コードブロック、リンク、画像、テーブルなどのフォーマットをボタンから簡単に挿入可能。
- **アウトライン表示**: ドキュメントの見出し構造からアウトラインを自動生成し、パネルに表示。
- **テーマ切り替え**: 目に優しいダークモードとライトモードをワンタップで切り替え。
- **文字数・行数カウント**: リアルタイムでステータスバーにドキュメントの行数、文字数、カーソル位置を表示。

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript (ES Modules)
- **ビルドツール**: [Vite](https://vitejs.dev/)
- **エディター**: [CodeMirror 6](https://codemirror.net/)
- **マークダウンパース**: [marked](https://marked.js.org/)
- **コードハイライト**: [highlight.js](https://highlightjs.org/)
- **図のレンダリング**: [Mermaid](https://mermaid.js.org/)

## ソース構成

```
src/
├── main.js         # アプリケーションのエントリーポイント・全体制御
├── editor.js       # CodeMirrorエディターの初期化・設定
├── preview.js      # マークダウンプレビュー・Mermaidレンダリング
├── toolbar.js      # ツールバーのボタン操作
├── outline.js      # アウトライン（見出し一覧）の生成・表示
├── file-manager.js # ファイルの読み込み・保存処理
├── theme.js        # ダーク/ライトモードの切り替え
└── style.css       # アプリケーション全体のスタイル
```

## 開発環境のセットアップ

### 前提条件

- [Node.js](https://nodejs.org/) がインストールされていること

### インストールと起動

1. プロジェクトディレクトリのルートに移動し、依存関係をインストールします。

```bash
npm install
```

2. 開発サーバーを起動します。

```bash
npm run dev
```

3. コンソールに表示されたURL（例: `http://localhost:5173`）にブラウザでアクセスします。

## ビルド手順

本番向けのデプロイメント用にビルドする場合は、以下のコマンドを実行します。

```bash
npm run build
```

`dist` ディレクトリに、最適化されミニファイされた静的ファイルが出力されます。
